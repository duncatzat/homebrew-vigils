# Vigils Homebrew Tap

Local-first control plane for AI agents — see what they do, approve what matters, keep secrets out.

## CLI

```bash
brew install duncatzat/vigils/vigil-hub
vigil-hub demo --tamper     # 30s falsifiable demo — no account, no network, no LLM
vigil-hub setup --all       # protect every detected agent (reversible)
```

## Desktop app (macOS)

```bash
brew install --cask duncatzat/vigils/vigils
```
